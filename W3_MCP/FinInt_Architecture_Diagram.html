End-to-End Flow: Research Request

1
User Action
User searches "RELIANCE" on Research page. Frontend calls POST /api/tool/cross_reference_signals with Bearer JWT.
2
Auth Pipeline
JWT validated via Keycloak JWKS → scope check (research:generate = Analyst) → rate limit check → audit log written.
3
CrewAI Kickoff
4 agents run in parallel: price (Angel One), fundamentals (Alpha Vantage), news (Finnhub), macro (RBI DBIE).
4
Data Facade
Each agent's tool calls hit DataFacade: L1→L2→source→fallback. Circuit breakers protect each API independently.
5
Synthesis
Synthesizer agent combines all findings into CrossSourceAnalysisOutput with signals, contradictions, citations.
6
Trust Score
Deterministic trust scorer: normalize signals → detect conflicts → compute 0-100 score. No LLM used.
7
Cache & Return
Result cached in Postgres (1hr TTL). JSON response with data, source, trust_score, disclaimer returned to frontend.
8
UI Render
Dashboard shows signal cards, Trust Score Panel (confirmed/conflict/missing), contradiction details, and source citations.
